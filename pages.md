# 博客搭建指导

博客代码提交操作流程:

gitbook流程

```
    gitbook通过本地编辑,可以自动将文件push到gitbug.

    gitbook仓库克隆地址,只支持https方式克隆

        https://git.gitbook.com/dongfangyu1/mybook.git
        
    本地编译生成html
    
    
github

    github只保存gitbook生成的html

    原始md文件不保存



码云    

    码云只保存gitbook生成的html

    原始md文件不保存
```
